# Node Blank

An infinite canvas application for creating math equations, markdown text, graphs, and images, built with vanilla JavaScript following Atomic Design Methodology.

[![fgjnhfgcv.jpg](https://i.postimg.cc/br5WzVbH/fgjnhfgcv.jpg)](https://postimg.cc/JHbPxqpG)

## üöÄ Features

- **Math Nodes**: Create and edit mathematical equations using MathLive
- **Text Nodes**: Write markdown with LaTeX support
- **Graph Nodes**: Plot mathematical functions
- **Image Nodes**: Upload and place images
- **Infinite Canvas**: Pan, zoom, and organize freely
- **Multiple Canvases**: Create, manage, and switch between multiple canvases
- **Persistence**: Auto-save to IndexedDB
- **Export/Import**: Save and load individual canvases as JSON
